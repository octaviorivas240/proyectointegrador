# Proyecto Integrador - Procesamiento Digital de Señales

**Nombre:** RIVAS TORRES OCTAVIO ALBERTO  
**Número de actividad:** PROYECTO 
**Fecha:** 11 de diciembre de 2025

## Requisitos del proyecto cumplidos
- Captura de foto, audio y video en tiempo real  
- Aplicación de FFT/IFFT  
- Convolución con filtros: Roberts, Prewitt, Sobel (H,V,ambos), Laplace, LoG  
- Convolución de audios + reproducción de los 3 sonidos  
- Muestreo (downsampling) de imágenes y audio  
- Interfaz gráfica con menús por tipo de señal  
- Reporte en PDF + capturas de pantalla  
- Código fuente + ejecutable
